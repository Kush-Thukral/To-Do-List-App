# ✅ To-Do List App (Tkinter)

A clean, minimal To-Do List App built with Python's Tkinter library.

### 🎯 Features:
- Add and delete tasks
- Sort tasks A–Z / Z–A
- Count tasks
- Random task picker
- Dark minimal UI (Nord theme inspired)

### 🛠 Built With:
- Python 3.x
- Tkinter (no external dependencies)
